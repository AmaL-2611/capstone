# capstone

## Repository name
capstone

## Project name
Django Dealer Review Platform

## Overview
This repository contains a Django backend, a React registration component, and static HTML pages for About and Contact. It includes sample API endpoints for dealers, reviews, authentication, car makes, and sentiment analysis.

## Structure
- `server/` - Django project, API endpoints, static site files
- `server/frontend/` - React frontend skeleton and static HTML pages
- `django_server` - sample terminal output showing the Django server running
- `loginuser`, `logoutuser`, `getdealerreviews`, `getalldealers`, `getdealerbyid`, `getdealersbyState`, `getallcarmakes`, `analyzereview` - sample cURL command files
- `CI/CD` - sample GitHub Actions workflow output
- `deploymentURL` - placeholder deployment URL
